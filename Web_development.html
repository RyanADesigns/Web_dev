Problems with RWD

The Responsive design solves a number of problems that stem from when the internet was created. Responsive Web Design makes the internet accesible to a mobile Audience. Responsitivity helps accesibility and makes "snap-to" styling to some degree obselete. With the multiplicity of devices. that rule the media responsive web design levels the playing field for both developers and those accessing the internet. one of my biggest problems as a upcoming developer is that I often style my page so that it looks good in whatever browser window I have open, The problem being it always looks bad when I put up my coded project up on another browser/ computer. With responsive design I can design a page that looks good no matter what device or Browser. Before responsive design, developers would have to create multiple pages of, semantically, the same content adusted each version to suite the target devices. With Responsive design all pages can be made off one scource code. In years to come developers will ask how they ever got by without Responsive Web design.
